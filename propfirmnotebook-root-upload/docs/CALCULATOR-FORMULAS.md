# Calculator Formulas

This document explains the first five calculators in plain language.

## Trailing Drawdown Calculator

The calculator estimates the drawdown floor.

For trailing drawdown:

```text
drawdown floor = highest account value - maximum drawdown amount
```

For static drawdown:

```text
drawdown floor = starting balance - maximum drawdown amount
```

Account cushion is:

```text
current equity - drawdown floor
```

## Position Size Calculator

Dollar risk is:

```text
account size x risk percentage
```

Estimated size is:

```text
dollar risk / stop risk per unit
```

## Daily Loss Limit Calculator

Current day result is:

```text
closed profit or loss + open profit or loss
```

Firm room left is:

```text
daily loss limit + current day result
```

## Prop Firm Challenge Planner

Profit target amount is:

```text
account size x profit target percentage
```

Average daily goal is:

```text
profit target amount / planned trading days
```

## Evaluation Progress Tracker

Progress to target is:

```text
current profit / target amount
```

Drawdown room is:

```text
current account value - drawdown floor
```

These tools are educational and are not financial advice.
