# PropFirmNotebook

This is the ready-to-upload version of the PropFirmNotebook website for investingonlineforbeginners.com.

The current GitHub repo name is expected to be `prop-firm-tools`. PropFirmNotebook is the public site brand.

It includes five educational tools:

- Trailing Drawdown Calculator
- Position Size Calculator
- Daily Loss Limit Calculator
- Prop Firm Challenge Planner
- Evaluation Progress Tracker

The site also includes:

- Homepage Q&A for prop firm and beginner trader searches
- Short FAQ sections on every tool page
- FAQ structured data for search engines
- Calculator formula documentation
- A question map for AI Overview-style discovery

The site is plain HTML, CSS, and JavaScript. It does not need a database or a paid app server.

The content rule is simple: no long articles. Keep answers short, direct, and useful.
